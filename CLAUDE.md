# CLAUDE.md

## Project Overview

This project is a web application for managing and designing PostgreSQL databases. It includes features like a custom text format for DB schemas, conversion from text to ERD (Entity-Relationship Diagram), SQL to text format, export/import functions, and AI-assisted schema creation/design. The app aims to provide an intuitive interface for developers to visualize, edit, and generate database schemas efficiently.

## Tech Stack

### Frontend

- **React** (v18.3.1 or latest): For building the user interface.
- **Vite** (v5.4.1 or latest): Fast build tool and development server.
- **shadcn/ui** (latest): Component library for accessible, customizable UI elements.
- **Tailwind CSS** (v3.4.10 or latest): Utility-first CSS framework for styling.

### Backend

- **Node.js** (v20.11.0 or latest LTS): Runtime environment.
- **Nest.js** vs **React Server Function**
- **Additional libraries**:
  - `node-sql-parser`: For parsing SQL to your custom text format.
  - `pg`: For direct Postgres connections (e.g., schema introspection).
  - `openai`
  - `mermaid`: For generating ERD diagrams from text (server-side rendering if needed).
  - `zod`: For schema validation and type safety.

### Other Tools

- **AI Integration**: Use OpenAI. This requires an API key stored in environment variables.
- **ERD Rendering**: `react-flow` for interactive diagrams on the frontend (alternative: `mermaid-react` for simpler static renders).
- **Parser for Custom Text Format**: Implement a custom parser using `peggy` (PEG.js successor) or `nearley` for defining and parsing your DSL.
- **Version Control**: Git.
- **Testing**: Jest for unit tests, Cypress for E2E.
- **Linting/Formatting**: ESLint + Prettier.

If you need a database for the app itself (e.g., to store user sessions or saved schemas), use PostgreSQL with Prisma ORM for type-safe queries. Otherwise, the app can be stateless, storing data in-memory or via local files. The application will be stateless in first stage.

This stack is chosen for its speed, scalability, and ease of integration. React + Vite ensures hot module replacement for dev productivity. Shadcn/ui + Tailwind provides modern, customizable components without heavy dependencies. Node/Express handles backend logic securely. If you prefer Python for AI (e.g., with LangChain), we can switch to FastAPI—let me know.
